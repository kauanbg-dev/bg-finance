# 💰 BG Finance

Sistema de controle financeiro feito com Node.js, Express e SQLite.

🔗 Deploy: https://bg-finance-delta.vercel.app/

---

## 📌 Sobre o projeto

O BG Finance é um sistema simples para controle de receitas e despesas, mostrando saldo total e resumo mensal.

---

## 🚀 Tecnologias usadas

- Node.js
- Express
- SQLite
- HTML
- CSS
- JavaScript
